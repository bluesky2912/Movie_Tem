# Movie_Tem
An interactive full-stack web application that helps users discover movies based on how their soul feels tonight. Built with PHP, MySQL, JavaScript, and the TMDB API.

## ✨ Features
- **Dynamic Mood Matching:** Browse movies curated specifically for your current vibe (Horror Night, Mind-Bending, etc.).
- **🎲 Surprise Me Button:** Instantly cycles through moods to select a randomized stream when caught in choice paralysis.
- **📌 Persistent Watchlist:** Live-toggle tracking synced cleanly with a custom normalized relational MySQL backend.

## 🛠️ Setup Instructions
1. Clone this repository.
2. Import `schema.sql` into your MySQL database server (e.g., via phpMyAdmin).
3. Copy `config/database.php.example` to `config/database.php` and fill in your local database credentials.
4. Run via local server environments like XAMPP or MAMP.
